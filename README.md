# :hammer: Build Setup

:heavy_plus_sign: Install dependencies
```
$ npm install # Or yarn install
```
:wrench: Create config/config.env 
```
PORT = 3000
MONGO_URI = 
GOOGLE_CLIENT_ID = 
GOOGLE_CLIENT_SECRET = 
```
:rocket: Run at localhost:3000
```
$ npm run dev
```
